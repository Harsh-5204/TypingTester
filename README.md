# ⌨️ Typing Tester

Typing Tester is a simple and interactive web application to measure typing speed, accuracy, and performance in real time. Built with **HTML**, **CSS**, and **JavaScript**, it helps users improve their typing skills through live feedback and performance stats.

## 🚀 Features

- 🧠 Random sentence loading for each test
- 🎯 Real-time character matching with instant feedback (green/red)
- ⏱️ Typing speed in WPM (Words Per Minute)
- 📉 Accuracy percentage tracking
- ❌ Prevents copy-paste to encourage genuine typing
- 🔁 Reset and try again functionality
